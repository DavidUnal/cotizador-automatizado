# 📊 Cotizador Automatizado e Inteligente (Excel & Google Sheets)

### 🎯 El Problema
En procesos comerciales técnicos, la elaboración manual de cotizaciones consume tiempo valioso y es propensa a errores de cálculo, lo que retrasa el cierre de ventas y afecta la imagen profesional ante el cliente.

### 💡 La Solución
Desarrollé una herramienta híbrida que automatiza la generación de propuestas comerciales técnicas. Utiliza lógica condicional avanzada y programación (VBA para Excel / Apps Script para Sheets) para estandarizar la captura de requerimientos.

### 🚀 Funcionalidades Clave
- **Generación Automática de PDF:** Exportación de la cotización con un solo clic.
- **Envío Masivo de Correos:** Integración con Gmail/Outlook para enviar la propuesta directamente al cliente.
- **Base de Datos Centralizada:** Registro automático de cada cotización en una hoja histórica para seguimiento comercial (Pipeline).
- **Cálculo de Márgenes en Tiempo Real:** Validación financiera inmediata para asegurar la rentabilidad de cada oferta.

### 🛠️ Tecnologías utilizadas
- **Google Apps Script / JavaScript:** Para la lógica en la nube.
- **VBA (Visual Basic for Applications):** Para la versión de escritorio.
- **SQL / Query Functions:** Manejo de bases de datos de productos y precios.
